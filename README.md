# Edge AI Vibration Monitor

## Predictive Maintenance with TinyML

This project demonstrates how vibration data collected from sensors can be used to detect anomalies and predict equipment failures using machine learning models deployed directly on embedded systems.

---

## Objective

Develop a low-cost Edge AI solution capable of:

* Acquiring vibration data
* Processing signals locally
* Detecting abnormal behaviour
* Running inference directly on embedded hardware
